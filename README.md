# Blender 2D Facial Rig Lip-Syncing System

# How to Use
1. Open up a graph editor
2. Click channels -> Sound to Samples
3. Sound to samples -> Samples to Keys
4. Go to Scripts
5. Input the proper object names into the variables
6. Run the script
